Detailed Description

An abstract specification for scanning a particular icon set type.

By extending this class and overwriting the abstract functions one can define strategies for parsing and accessing different icon sets. This is a good point to join development: Define a scanner strategy for a particular icon set type / structure / format you would like to use with Icon Set Forge.
