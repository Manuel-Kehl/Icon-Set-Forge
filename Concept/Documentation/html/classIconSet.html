Detailed Description

The model representation of an icon set.

Contains the set of icons, their corresponding classifications and an AbstractScannerStrategy used to load the icon set. All data is owned by the IconSet instance and therefore its lifetime bound to the scope of that class. To realise this, std::unique_ptrs are being used for members.
